1. JavaScript

JavaScript is a high-level, dynamic, interpreted programming language that is used to create interactive web pages and web applications. It was created in 1995 by Brendan Eich while he was working for Netscape Communications Corporation. JavaScript is a client-side scripting language, which means it runs on the user's computer rather than on the server hosting the website.

JavaScript is used in a variety of ways, from simple webpage animations to complex web applications like Gmail and Google Maps. It is also used for server-side programming with platforms like Node.js. JavaScript is supported by all major web browsers, including Chrome, Firefox, Safari, and Internet Explorer.

2. Variables and Data Types

JavaScript is a dynamic language with dynamic types. Variables in JavaScript are not directly associated with any particular value type, and any variable can be assigned (and re-assigned) values of all types.

In Javascript, datatypes are divided into two categories: primitive and object.
Primitive datatypes are built-in to the language and include include numbers, strings, booleans, null and undefined.
Objects are created using constructors and have properties that can be accessed using dot notation (e.g., myObject.propertyName).

3.Type Coercion

Type coercion is the automatic or implicit conversion of values from one data type to another (such as strings to numbers). Type conversion is similar to type coercion because they both convert values from one data type to another with one key difference — type coercion is implicit whereas type conversion can be either implicit or explicit.

There are some downsides to this type of coercion: It can lead to unexpected results in your code; It can cause performance issues if you use it improperly, since it involves extra work by the computer; It can introduce bugs into your code if you're not careful about which types are being coerced and how they should be handled.

4. Loose vs. Strict Equality Operator

In JavaScript, there's both the double equality operator (== which is called the loose equality operator) and the triple equality operator (=== which is called the strict equality operator). You use both operators to compare values' equality.

The loose equality operator does a loose check. It checks if values are equal. The types are not a focus for this operator - only the values are the major factor. 

    const variable1 = 20;
    const variable2 = "20";

    console.log(variable1 == variable2);// true

When you use the loose equality operator with values of different types, what happens first is coercion. Again, this is where one value is converted to the type that fits the other, before the comparison occurs.
In this case, the string "20" is converted to a number type (which is 20) and then compared with the other value, and they are both equal.

The strict equality operator does a strict check – that is, it strictly checks the values compared, as well as the types. Type coercion does not occur here, so there are no unexpected answers. 

    const variable1 = 20;
    const variable2 = "20";
            
    console.log(variable1 === variable2);// false

5. Strict Mode

Strict mode is a feature in JavaScript that allows developers to opt-in to a stricter version of the language, which makes it easier to write secure and optimized code. According to MDN Web Docs, strict mode makes several changes to normal JavaScript semantics: it eliminates some JavaScript silent errors by changing them to throw errors, and it fixes mistakes that make it difficult for JavaScript engines to perform optimizations. This means that strict mode code can sometimes be made to run faster than identical code that's not in strict mode.

To enable strict mode, you can add the string "use strict" ; at the beginning of a script or a function. If declared at the beginning of a script, it has global scope (all code in the script will execute in strict mode).

6. Functions

Functions in JavaScript are treated as first-class objects, which means they can be passed around like any other value. They can be assigned to variables, passed as arguments to other functions, and returned as values from functions. In JavaScript, functions can be declared using the function keyword followed by the function name, or they can be expressed as anonymous functions. 

    //Function Syntax
    function greet(name) {
        console.log("Hello, " + name + "!");
    }
              
    greet("John"); // Output: Hello, John!

In the example above, we declare a function called greet that accepts a parameter called name. Inside the function body, we log a greeting message to the console using the provided name. We then call the function and pass in the argument "John". 

    //Anonymous Function Syntax
    var sum = function(a, b) {
        return a + b;
    }
                  
    console.log(sum(2, 3)); // Output: 5

In the example above, we declare an anonymous function and assign it to a variable called sum. The function takes two parameters a and b and returns their sum. We then call the function and pass in the arguments 2 and 3 to get the sum 5

7.Callback Functions

In JavaScript, a callback function is a function that is passed as an argument to another function and is executed inside that function.

One common use case for callback functions is handling asynchronous operations, such as making an API call or fetching data from a database. Instead of waiting for the operation to complete before moving on to the next line of code, we can pass a callback function to the asynchronous function which will be executed once the operation has completed.

Here's an example of a callback function being used with an asynchronous operation: 

    function greeting(name, callback) {
        console.log("Hello, " + name + "!");
        callback();
    }
              
    function sayGoodbye() {
        console.log("Goodbye!");
    }
              
    greeting("John", sayGoodbye);

In this example, we have two functions greeting and sayGoodbye. greeting takes two parameters: name and callback. It logs a greeting message to the console using the provided name and then calls the callback function.

sayGoodbye is the callback function, it logs a goodbye message to the console.

We call greeting and pass in the arguments "John" and sayGoodbye. When greeting is called, it logs "Hello, John!" to the console and then calls the callback function, which in this case is sayGoodbye.

The output of this code will be: 

            Hello, John!
            Goodbye!

8.Loops

A loop is a programming construct that allows you to execute a block of code repeatedly until a certain condition is met. The most common types of loops in JavaScript are for loops, while loops, and do-while loops.

A for loop is used when you know how many times you want to execute the loop. Here is an example: 

    for (let i = 0; i < 5; i++) {
        console.log("Hello, world!");
    }

A while loop is used when you want to execute a block of code while a certain condition is true. Here is an example: 

    let i = 0;
        while (i < 5) {
        console.log("Hello, world!");
    }

Now, in addition to these basic loops, JavaScript also has a number of loop methods that can be used with arrays. These methods include forEach(), map(), and filter().

• forEach()
This method is used to loop through the elements of an array and execute a function for each element 

    let numbers = [1, 2, 3, 4, 5];

    numbers.forEach(number => {
        console.log(number * 2);
    });

• map()
This method is used to create a new array with the results of calling a provided function on every element in the calling array 

    let numbers = [1, 2, 3, 4, 5];

    let doubledNumbers = numbers.map(function(number)  {
        return number * 2;
    });
                
    console.log(doubledNumbers);

• filter()
The filter() method creates a new array with all elements that pass the test implemented by the provided function 

    let numbers = [1, 2, 3, 4, 5];

    let evenNumbers = numbers.filter(function(number) {
        return number % 2 === 0;
    });
                
    console.log(evenNumbers);

9. Date and Time

In JavaScript, the Date() function is a built-in object that allows you to work with dates and times. According to Mozilla Developer Network, "A JavaScript date is fundamentally specified as the number of milliseconds that have elapsed since the ECMAScript epoch, which is defined as the midnight at the beginning of January 1, 1970, UTC (equivalent to the UNIX epoch)".

In simpler terms, the Date() function allows you to create a JavaScript object that represents a specific date and time, which you can then manipulate or format as needed. The Date() function has several methods that allow you to get or set specific parts of a date (such as the year, month, day, hour, minute, second, and millisecond).

• To get the current date and time, create a new Date() object with no arguments: 

    let currentDate = new Date();
    console.log(currentDate);
    
    //ouptut
    Wed Oct 06 2021 18:25:42 GMT-0400 (Eastern Daylight Time)

• To get the current year, use the getFullYear() method: 

    let currentYear = new Date().getFullYear();
    console.log(currentYear);
    //ouptut
    2021

• To get the current month (as a number from 0-11), use the getMonth() method: 

    let currentMonth = new Date().getMonth();
    console.log(currentMonth);

    //ouptut
    9

• To get the current day of the month (as a number from 1-31), use the getDate() method: 

    let currentDay = new Date().getDate();
    console.log(currentDay);

    //ouptut
    6

• To get the current day of the week (as a number from 0-6, where 0 is Sunday and 6 is Saturday), use the getDay() method: 

    let currentDayOfWeek = new Date().getDay();
    console.log(currentDayOfWeek);

    //ouptut
    3

10. fetch()

fetch() is a built-in function in JavaScript that allows you to make network requests to retrieve resources from the server.

Here's an example of how to use fetch(): 

    fetch('https://anyUrl')
        .then(response => response.json())
        .then(data => console.log(data));

The 'fetch()' function returns a Promise, which we can then use to retrieve the response data.

The first 'then()' method is used to convert the response object to JSON format. The second 'then()' method logs the JSON data to the console.

You can also use 'fetch()' to make requests with different options, such as adding headers or sending a POST request with data.
